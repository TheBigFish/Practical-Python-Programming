## 描述

用于高效循环的迭代函数

这个模块中提供的函数具有和"lazy functional programming language" Haskell 和 SML 相似的特点. 他们都是为了跑得更快和更有效的使用内存. 但他们也被牵扯在一起以表示更为复杂的迭代算法.
